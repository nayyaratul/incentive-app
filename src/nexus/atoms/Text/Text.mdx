# Text

General-purpose text component for body copy, captions, and supporting content. Pair with `Heading` for a complete type system.

<Primary />

<Controls />

## Variants

Each variant targets a different content role.

- **body** — Standard paragraph text. Renders as `<p>`.
- **bodyLg** — Larger body copy for introductions or lead paragraphs. Renders as `<p>`.
- **caption** — Secondary info like timestamps or helper text. Renders as `<span>`.
- **micro** — Small metadata such as progress counters. Renders as `<span>`.
- **overline** — Uppercased category labels above headings. Renders as `<span>`.

## Accessibility

- Defaults to semantic `<p>` for body variants and `<span>` for inline variants.
- Use the `as` prop to render a different element when semantics require it.
- The `truncate` prop clips text with an ellipsis — ensure the full content is available via `title` attribute or tooltip when truncating.

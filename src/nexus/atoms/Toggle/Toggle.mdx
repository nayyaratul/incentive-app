# Toggle

A two-state button built on Radix UI's Toggle primitive. Use it when users need to switch a single option on or off, such as bold formatting or a filter.

<Primary />

<Controls />

## Variants

- **default** — subtle background change on press.
- **outline** — bordered style, good for toolbar buttons.
- **pill** — rounded, tag-like appearance for filter chips.

## Sizes

Three sizes: `sm`, `md`, and `lg`.

## Accessibility

- Built on `@radix-ui/react-toggle`, which manages `aria-pressed` automatically.
- Provide `aria-label` for icon-only toggles.
